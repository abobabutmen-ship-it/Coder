Тут находятся пустые шаблоны переводов для удобства переводчиков.

Использование:
- Откройте соответствующий файл, например `locales/fr.template.json`.
- Замените значения "TODO: translate" на корректные переводы.
- Переименуйте файл в `locales/<код_языка>.json` (например `fr.json`) или создайте PR с обновлением соответствующего файла.

Замечание: CI валидатор проверяет соответствие ключей между файлами `locales/*.json`. Шаблоны имеют те же ключи и облегчают работу переводчикам.
